咒語參考

不複製canvas也可以試試自己玩一次！

請你幫我用 HTML、Tailwind CSS 和原生 JavaScript 建立一個單頁式的網頁應用程式，叫做「課文簡化學習單系統」。這個系統能讓使用者貼上課文，然後透過 AI 自動生成各段摘要、課文圖解和隨堂測驗。

詳細功能分解 (Detailed Breakdown)

請嚴格按照以下規格來實現：

1. 介面與外觀 (UI/UX):

  • 整體風格: 使用 Tailwind CSS 進行排版和美化，風格要簡潔、現代，卡片式設計搭配圓角和陰影。
  • 字體: 引入 Google Fonts 的「Noto Sans TC (思源黑體)」作為全站字體。
  • 主要輸入區: 一個大的 textarea 讓使用者貼上課文，一個主要的「生成學習單」按鈕。
  • 結果呈現區: 分成「各段摘要」、「課文圖解」、「隨堂測驗」、「答案與解析」四個卡片。
  • 互動按鈕: 每個區塊都要有「重新生成」的按鈕，摘要和測驗區要有「複製內容」的按鈕。

2. JavaScript 核心邏輯:

  • 不使用框架: 請用原生 JavaScript (Vanilla JS)。
  • API 呼叫: 設計函式呼叫 Google Gemini API 和 Imagen API。
  • 功能實現: 主流程、重新生成、錯誤處理、複製與下載功能。

3. 頁面文字:

所有使用者介面上的文字都必須使用繁體中文。
